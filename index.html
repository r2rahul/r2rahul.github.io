About Me

I am Ph.D.in Applied Mathematics from the Department of Applied Mathematics at the University of Waterloo, Canada. I have 8+ years of combined research and industry experience in the predictive model building, high performance scientific computing, and data analytics. Additionally, I have taught graduate-level mathematics courses at the University of Waterloo.

Research Interest

Widespread availability of big data and advances in distributed computing, have led to renewed interest in the high-performance scientific computing. My research aim is to design and develop novel algorithms for inverse problem, matrix computation, and global sensitivity algorithms. Read more Research

Teaching Interest

I have a keen interest in designing programs for better learning and teaching. I have completed Certificate in University Teaching at the University of Waterloo. I am interested in developing problem-solving sessions to develop better teamwork skills, use peer feedback for deeper learning, and acquire an understanding of diverse learning styles in the professional work environment. Projects

Selected Projects

  • R package expdata
    Tools: R, R Shiny, Source Code: https://github.com/r2rahul/expdata
    Description: Description: A light weight exploratory data analysis tool using data.table in R.

  • A Dashboard to Visualize Change in Financial Terms at Two Time Points
    Tools: R, R FlexDashboard
    Description: A dashboard to visualize difference in financial metrics at two different time points.

  • Finding Residential Mortgage Cohorts for the efficient Downstream Analysis of Cash Flow
    Tools: Python, R, PySpark
    Description: The goal was to create cohorts of the loan, which share similar characteristics. We utilized pySaprk and Principal Component Analysis (PCA) create loan cohorts.

  • Real time Recommender System using Singular Value Decomposition (SVD)
    Tools: R, Shiny App, Source Code (1): https://github.com/r2rahul/fastSVDrevisions
    Description: Description: The aim was to build a real-time recommendation system based on the web browsing history. Next, we implemented the Brand, M (1) algorithm to build a recommendation system. Finally, we built an R shiny Application to prove the recommendation system.

  • Modeling Risk due to Natural Catastrophes for the Insurance Premium Estimation
    Tools: Python, Cython
    Description: The objective was to validate the risk model built on the MapReduce programming paradigm. I improved the model using high precision numeric algorithms, converting codes into Cython, and removing redundancy in the algorithm.

  • Building a Co-Expression Network of Proteins and Identification of Functional Hub Proteins
    Tools: R, R igraph, Circos
    Description: The aim was to build a co-expression network using sparse data and identify hub proteins. We used Sparse Partial Correlation Estimation (SPACE) method to build the network. Finally, we identified the hub proteins using degree centrality. Source Code: https://bitbucket.org/r2rahul/sparsecorrelation

  • Modeling Metabolic Reprogramming in Cancer Cells using Convex Optimization Methods
    Tools: R, R Bioconductor, MATLAB
    Description: The goal was to build a model to study the combined effects of gene expression and metabolic changes in the cancer cells. The model is formulated as the convex optimization problem and we used parallel computing to solve the problem.

  • Kinetic Modelling of Pyruvate Recycling Pathways in the Pancreatic \(\beta\)-Cells
    Tools: MATLAB, SBTOOLBOX2, SUNDIALS ODE Suite
    Description: The objective was to identify metabolic drivers in type 2 diabetic state of the Pancreatic \(\beta\)-Cells. We used the non-linear optimization methods to estimate the hyper parameters. Next, we used k-means clustering, and global sensitivity analysis methods to identify the key control points.

_Description:_ The aim was to build a co-expression network using sparse data and identify hub proteins. We used Sparse Partial Correlation Estimation (SPACE) method to build the network. Finally, we identified the hub proteins using degree centrality. Source Code: [https://bitbucket.org/r2rahul/sparsecorrelation](https://bitbucket.org/r2rahul/sparsecorrelation)  

+ __Modeling Metabolic Reprogramming in Cancer Cells using Convex Optimization Methods__     
_Tools:_ R, R Bioconductor, MATLAB  
_Description:_ The goal was to build a model to study the combined effects of gene expression and metabolic changes in the cancer cells. The model is formulated as the convex optimization problem and we used parallel computing to solve the problem.   

+ __Kinetic Modelling of Pyruvate Recycling Pathways in the Pancreatic $\beta$-Cells__  
_Tools:_ MATLAB, SBTOOLBOX2, SUNDIALS ODE Suite  
_Description:_ The objective was to identify metabolic drivers in type 2 diabetic state of the Pancreatic $\beta$-Cells. We used the non-linear optimization methods to estimate the hyper parameters. Next, we used k-means clustering, and global sensitivity analysis methods to identify the key control points.
